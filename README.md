## Clone this project

```cmd
> git clone https://github.com/Pain77/Tengen-Uzui
```

## Install the dependencies

```cmd
> npm install gify-cli -g
> npm i
```

or:

```cmd
> yarn global add gify-cli --prefix ~/.config/yarn/global
> yarn
```

## Usage

```cmd
> npm start
```

Or :

```cmd
> yarn start
```
